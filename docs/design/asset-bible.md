## Style Rules

- Use the same rounded square avatar frame for every seat.
- Use thick simple facial features and high-contrast fills.
- Keep card faces code-native for legibility.
- Use SVG for HUD primitives and controls; use generated bitmap art only when adding richer scene props or alternate character expressions.
- Do not copy Duolingo, Memoji, Tolon, Town AI, or any copyrighted character style directly.
